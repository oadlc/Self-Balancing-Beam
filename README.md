# Self-Balancing-Beam
A closed-loop control system using a servo motor to tilt a beam and keep a rolling ball centered, driven by a PID controller and position sensor.

3D Parts - A CAD directory that has the SLDPRT files for each component needed for the project. Each SLDPRT file was made on SOLIDWORKS
PID_schematic.png - A picture of the schematic for the circuit: includes the MG966R, and the GP2Y0A21YK0F Sharp IR sensor.
SRC.zip - This is where the .ino script for the PID controlled system is.
